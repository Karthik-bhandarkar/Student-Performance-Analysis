# Student Performance Analysis

## Overview
This project analyzes student performance data to understand how various factors such as study habits, socioeconomic status, and parental involvement influence exam scores.

## Features
- **Data Cleaning**: Handles missing values and inconsistences.
- **Data Visualization**: Uses Seaborn and Matplotlib to explore relationships.
- **Analysis**: Examines correlations between variables like `Hours_Studied`, `Attendance`, and `Exam_Score`.

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the analysis in `Student.ipynb`.

## Dependencies
- pandas
- seaborn
- matplotlib
- openpyxl
